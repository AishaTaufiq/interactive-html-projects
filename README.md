# Interactive HTML Projects

A collection of interactive HTML, CSS, and JavaScript projects developed as part of my Artificial Intelligence coursework. These projects were created to explore AI concepts through interactive visualizations, simulations, and web-based applications.

## Projects

### 🏠 Smart Room Digital Twin

An interactive smart room simulation that visualizes environmental conditions and smart device controls through a modern dashboard interface.

**Features**

* Interactive smart room visualization
* Environmental monitoring
* Device control simulation
* Responsive user interface

---

### 🐦 Cuckoo Search Algorithm Simulator

An educational visualization of the Cuckoo Search optimization algorithm, demonstrating how candidate solutions evolve to find optimal results.

**Features**

* Interactive algorithm simulation
* Step-by-step visualization
* Fitness tracking
* Educational explanations

---

### ⚛️ Particle Swarm Optimization Simulator

A visual simulator demonstrating the Particle Swarm Optimization (PSO) algorithm, showing how particles cooperate to locate the best solution within a search space.

**Features**

* Real-time particle animation
* Convergence visualization
* Algorithm learning panel
* Interactive controls

---

### 🚀 Galaxy Defender Plus

A browser-based arcade game featuring multiple difficulty levels, achievements, score tracking, particle effects, and responsive controls.

**Features**

* Multiple difficulty modes
* Wave-based gameplay
* Achievement system
* Score and combo tracking
* Keyboard and touch controls

---
